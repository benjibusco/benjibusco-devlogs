### Repository Adapter Architecture

Migrated the Projects frontend away from direct localStorage usage by
introducing a repository adapter layer.

Added:

```text
projects-config.js
projects-api.js
projects-local-storage.js
projects-data.js
```

The page scripts now interact only with a shared data layer while the
repository selects either:

- API
- localStorage

The adapter uses the shared environment-aware API helper so development,
staging, and production automatically target the correct backend.

## Frontend Architecture

Both Projects and Tasks now share the same scalable frontend
architecture:

```text
Page Scripts
        │
        ▼

        │
 ┌──────┴──────┐
```

data source while providing a consistent foundation for future Freelance
Workspace modules.
