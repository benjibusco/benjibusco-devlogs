## Apache Reverse Proxy Fix

Updated Apache SSL virtual host configuration to correctly forward HTTPS proxy headers.

Added:

```apache
RequestHeader set X-Forwarded-Proto "https"
RequestHeader set X-Forwarded-Ssl "on"
```

Enabled proper proxy-aware HTTPS handling for Express sessions.

Validated Apache configuration:
