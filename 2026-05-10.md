### JavaScript Architecture

Created `/scripts/status.js` to isolate dashboard logic and prevent `main.js` from becoming bloated.

Established a more modular frontend architecture with improved scalability and feature separation.

Moved scripts into the document `<head>` using `defer`:

```html id="m3c6pr"
<script src="/scripts/main.js" defer></script>
<script src="/scripts/status.js" defer></script>
```

This enabled non-blocking rendering, preserved script execution order, and improved DOM safety for larger frontend systems.
